# AI-Powered-Climate-Awareness-Action-Assistant

### Project Overview

This project presents a prompt-based conversational AI assistant designed to support climate awareness and sustainable decision-making. The solution helps users understand climate change issues, sustainability practices, and environmentally responsible actions through simple, guided conversations.

The system uses Natural Language Processing (NLP) and a conceptual Agentic AI workflow to interpret user queries, guide decision paths, and provide responsible, transparent responses related to climate and sustainability.
